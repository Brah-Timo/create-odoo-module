# Example: task-tracker

This is a **complete, working example** of an Odoo module generated by `create-odoo-module`.

## How it was generated

```bash
npx create-odoo-module task-tracker \
  --with-api \
  --with-tests \
  --with-reports \
  --with-wizard \
  --with-owl \
  --odoo-version 17 \
  --author "Example Corp" \
  --category "Project" \
  --depends "project" \
  --no-interactive
```

## What's included

| Path | Description |
|------|-------------|
| `task-tracker/odoo_module/` | Full Odoo Python module |
| `task-tracker/odoo_module/models/` | `task_tracker.py` — model with full state machine |
| `task-tracker/odoo_module/views/` | Form, List, Kanban, Search, Activity, Menus |
| `task-tracker/odoo_module/controllers/` | REST API controller (all CRUD endpoints) |
| `task-tracker/odoo_module/security/` | ACL CSV + security groups XML |
| `task-tracker/odoo_module/data/` | Auto-sequence record |
| `task-tracker/odoo_module/tests/` | 10 pytest tests |
| `task-tracker/odoo_module/wizard/` | Bulk action wizard (TransientModel) |
| `task-tracker/odoo_module/report/` | QWeb PDF report |
| `task-tracker/odoo_module/static/` | OWL component JS + SCSS |
| `task-tracker/odoo_module/i18n/` | Translation template (`.pot`) |
| `task-tracker/scripts/` | Docker Compose + deploy scripts |
| `task-tracker/.env.example` | Environment variable template |
| `task-tracker/package.json` | npm scripts for deploy/dev/test |

## Quick start

```bash
cd task-tracker
cp .env.example .env            # fill in ODOO_URL, ODOO_DB, ODOO_PASSWORD
npm run dev                     # start Odoo + PostgreSQL via Docker
# In another terminal:
npm run deploy                  # upload and install the module
```
